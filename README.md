This project analyzes sales data from Bright Coffee Shop to identify key trends, performance drivers, and business insights. The analysis focuses on revenue, transactions, product performance, store locations, and time-based sales patterns.

The goal of this project is to support data-driven decision-making by presenting insights through clear visualizations and a structured dashboard.
